# Usage

```
go get -u github.com/zieckey/etcdsync

go get -u github.com/fzzy/radix/redis
go get -u github.com/amyangfei/redlock-go

go get -u github.com/hashicorp/consul/api

bash test.sh nodlm
bash test.sh etcd
bash test.sh redis
bash test.sh consul
```
